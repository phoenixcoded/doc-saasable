# Analytics

Setting up Google Analytics in SaasAble is very simple. You need to update a tag ID in .env:

{% code title=".env" %}
```properties
...

## Google Analytics
NEXT_PUBLIC_ANALYTICS_ID=
```
{% endcode %}
